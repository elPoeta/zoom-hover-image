# Project Name
zoom-image

## Installation

import zoom-image.js component

<head>
    <script src="./zoom-image.js"></script>
</head>

## Usage
  <zoom-image 
        image="your url image"
        height='600'
    >
    </zoom-image>
 
   Attributes :
        image: url image
        height :  height display - default 400px
